Technologies Used

This application was developed using HTML, CSS, Bootstrap, JavaScript, Plotly.js, and Tensorflow.js.

The USDA quick stats service provided data on total crop yields.

The Multivariate Adaptive Climate Analogs (MACA) provided past, present, and future weather simulation data.

GitHub was used to host the data for querying within the application.

Neural Networks

Brief overview of how Tensorflow.js was used in this web application to train a neural network. More information can be found here.

1. Creates a sequential tensorflow model with an input and output layer.

2. Data is normalized and converted to tensors.

3. Then the model is compiled using the adam optimizer and mean squared error.

4. The batch size is 8 for the amount of months of data per year. The number of epochs is 50.

5. Then the model is tested, and here is where the yield data is predicted for the years 2021-2031.
